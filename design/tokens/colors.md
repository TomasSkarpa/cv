## Color tokens

Flat ink on paper. Light surfaces, near-black text, no gradients.

### Semantic roles

| Token | Value | Usage |
|-------|-------|-------|
| `color-bg-surface` | `#f7f7f5` | Page background |
| `color-bg-subtle` | `#ececea` | Grouped sections, footer |
| `color-bg-elevated` | `#ffffff` | Cards, form fields, header |
| `color-fg-default` | `#0a0a0a` | Primary text, links, buttons |
| `color-fg-muted` | `#5c5c5c` | Secondary text, metadata |
| `color-fg-inverse` | `#f7f7f5` | Text on dark surfaces |
| `color-accent-primary` | `#0a0a0a` | Links (underlined), primary buttons |
| `color-accent-primary-hover` | `#333333` | Hover state |
| `color-accent-secondary` | `#5c5c5c` | Section numbers, metadata accents |
| `color-border-subtle` | `#d4d4d0` | Dividers, card borders |
| `color-border-strong` | `#999993` | Emphasized borders |
| `color-border-critical` | `#b91c1c` | Validation errors |
| `color-bg-critical-subtle` | `#fef2f2` | Error field background |

### Contrast notes

- `color-fg-default` on `color-bg-surface`: ~18:1 (AAA)
- `color-fg-muted` on `color-bg-surface`: ~5.5:1 (AA)
- `color-fg-inverse` on `color-fg-default`: ~18:1 (AAA)
